The Goal

Score the most points by defecting or cooperating in matches of up to 8 people. This is an implementation of Axelrods tournaments.

Each round occurs independently. Each round you must decide whether to cooperate (C) or defect (D) against each active opponent. The pairwise interaction is scored and added to your current score. Outside of the first turn you will see how opponents treated you last round (whether they cooperated or defected against you) for reference.

  Rules

Game Setup:
  • The game is played over 60 turns
  • Multiple players compete simultaneously

Scoring (per pairwise interaction):
  • Both cooperate (C vs C): 4 points each
  • Both defect (D vs D): 1 point each
  • Defector vs Cooperator (D vs C): 7 points for defector, 0 points for cooperator
Your score per turn is the sum of payoffs against all active opponents.

Defeat conditions:
You are eliminated if you:
  • Output the wrong number of lines (must equal number of active opponents)
  • Use invalid format (each line must be "opponent_id move")
  • Reference yourself, non-existent, or eliminated players
  • Include duplicate opponent IDs
  • Use invalid moves (anything other than C or D)
  • Timeout
Eliminated players no longer participate and are removed from opponent lists.

  Example

Initial Input: 
0 2

Turn 1 Input (3 players total): 

1 N
2 N
You are Player 0. Players 1 and 2 have no previous moves (first turn).

Turn 1 Output: 
1 C
2 C
You cooperate with both opponents.

Turn 2 Input: 
2
1 C
2 D
Player 1 cooperated with you, Player 2 defected.

Scoring: If you played C against both, you earned 4 points (vs Player 1) + 0 points (vs Player 2) = 4 points that turn.
Initialization Input

Line 1: myId, your player ID (integer)

Line 2: initialOpponentCount, the initial number of opponents (integer)

Input for One Game Turn

Line 1: opponentCount, number of active opponents this turn (integer)

Next opponentCount lines: Two space-separated values:

  • opponentId: the opponent's ID (integer)
  • move: their previous move against you - 'C' (cooperate), 'D' (defect), or 'N' (no previous move on first turn)

Output for One Game Turn
opponentCount lines, one per active opponent, each containing two space-separated values:
  • opponentId: the opponent's ID
  • move: your move against them - C (cooperate) or D (defect)

Constraints
2 ≤ players ≤ 8
maximum number of turns = 60
Allotted response time per turn is ≤ 100 milliseconds.
